## EditorJ
**It is a text editor built with javafx. You can edit files or create new files.**

## Features
 - **Full-Screen**
 - **DarkMode**
 - **Change Font**
 - **Find and Replace**
 - **Change Background and Text Color**

## Requirement to run application
 - Java SE 8

## Modify or Add new Features

### Requirements
 - Gradle 
 - Java SE 8
 - IntelliJ

### How to build 

`$ ./gradlew build`

### How to run 

`$ ./gradlew run`
